# A3 Presentation Service

## Что это

Это сервис для автоматической сборки PowerPoint-презентаций из текстовых документов в фирменном стиле компании.

Пользователь загружает документ, а система:

- понимает его структуру
- выделяет заголовки, списки, таблицы и текстовые блоки
- подбирает подходящие типы слайдов
- формирует готовую `.pptx` презентацию

## Для чего нужен сервис

Сервис нужен для того, чтобы сократить ручную работу по сборке презентаций и сделать результат единообразным по стилю.

Он решает такие задачи:

- ускоряет подготовку презентаций из стратегических документов
- уменьшает ручную верстку в PowerPoint
- сохраняет фирменный визуальный стиль
- помогает собирать презентации по единым правилам

## Как работает

Общий сценарий работы выглядит так:

1. Пользователь загружает документ.
2. Система анализирует документ и понимает его структуру.
3. Система определяет:
   - где титульная часть
   - где обычный текст
   - где списки
   - где таблицы
4. Для каждого блока подбирается подходящий тип слайда.
5. На выходе формируется готовая презентация в фирменном шаблоне.

## Что система уже умеет

На текущем этапе сервис умеет:

- работать с документами `docx`
- выделять структуру документа
- строить титульный слайд
- собирать текстовые слайды
- собирать слайды со списками
- собирать слайды с таблицами
- сохранять единый корпоративный стиль презентации

## Что было сделано в проекте

В ходе работы были выполнены ключевые улучшения.

### 1. Настроена логика чтения документа

Система теперь различает:

- заголовки
- подзаголовки
- обычный текст
- bullet-списки
- таблицы

Это важно, потому что без этого документ воспринимается как сплошной текст, и презентация получается хаотичной.

### 2. Настроена логика выбора типа слайда

Система больше не просто добавляет текст на случайные слайды.
Теперь она подбирает тип слайда под содержание:

- титульный слайд
- текстовый слайд
- слайд со списком
- слайд с таблицей

### 3. Улучшена работа с таблицами

Было сделано:

- корректное извлечение таблиц из документа
- размещение таблиц в нужных секциях презентации
- адаптация размеров таблиц под слайд
- уменьшение лишних разрывов таблиц на несколько слайдов

### 4. Улучшена работа со списками

Система теперь:

- понимает, где в документе именно список, а не просто текст
- не превращает обычные списки в лишние карточки
- формирует настоящие bullet-списки в PowerPoint

### 5. Улучшен титульный слайд

Было доработано:

- корректное определение заголовка презентации
- правильное размещение титульного текста
- соответствие фирменному стилю
- корректный цвет текста на темном фоне

## Текущий результат

Сейчас сервис уже находится в рабочем состоянии для основного сценария:

- на вход подается бизнес-документ
- на выходе получается оформленная презентация в корпоративном стиле

Результат уже пригоден для дальнейшего использования и развития.

## Коротко

Проект уже решает основную задачу:

из текстового документа автоматически создается PowerPoint-презентация в фирменном стиле.

Главная ценность сервиса в том, что он не просто переносит текст в слайды, а пытается понять структуру документа и собрать презентацию осмысленно.
