# A3 Presentation Service One-Pager

## Идея

Это сервис, который автоматически превращает текстовые бизнес-документы в готовые презентации PowerPoint в фирменном стиле компании.

## Проблема

Сейчас подготовка презентаций из стратегических документов требует ручной работы:

- переносить текст в слайды
- выбирать подходящую структуру
- верстать таблицы и списки
- следить за единым визуальным стилем

Это долго, дорого и зависит от конкретного исполнителя.

## Решение

Сервис берет документ и автоматически:

- определяет структуру материала
- понимает, где заголовки, текст, списки, таблицы и изображения
- подбирает подходящие типы слайдов
- собирает презентацию в корпоративном шаблоне

## Ценность

Сервис дает бизнесу:

- сокращение времени на подготовку презентаций
- единый фирменный стиль
- снижение ручной верстки
- более предсказуемое качество результата

## Что уже работает

На текущем этапе система уже умеет:

- загружать документ
- разбирать его структуру
- выделять титульную часть
- строить текстовые слайды
- строить bullet-списки
- строить слайды с таблицами
- строить слайды с графиками
- строить слайды с изображениями
- генерировать итоговую `.pptx` презентацию

## Текущий статус

Основные рабочие сценарии уже реализованы:

`документ -> структура -> слайды -> готовая презентация`

Сервис находится не просто в состоянии рабочего MVP, а в состоянии рабочего pipeline с отдельным quality-contract слоем:

- backend unit/regression tests
- deck-level quality checks
- frontend smoke и visual checks

## Следующий этап

Дальнейшее развитие может идти в сторону:

- вынесения quality-contracts в отдельный CI gate
- расширения визуальных и deck-level quality правил
- улучшения финального визуального качества
- добавления предпросмотра и более удобного пользовательского интерфейса

## Короткий итог

Проект уже решает основную задачу:

автоматически превращает текстовый документ в презентацию PowerPoint в корпоративном стиле.
